# Miqat — стек и библиотеки (шпаргалка для разговора с разработчиками)

> Miqat — **нативное приложение для macOS** (виджет времён намаза в чёлке).
> **Одной фразой разработчику:** «Нативный macOS-виджет на Swift + SwiftUI, собран через Swift Package Manager, данные считаются на устройстве (без сервера), распространяется напрямую `.dmg`, подписан Developer ID и нотаризован».

---

## 1. Язык, платформа, требования
| | |
|---|---|
| **Язык** | Swift |
| **Платформа** | macOS **13+** (Ventura и новее) |
| **Архитектура CPU** | Apple Silicon (arm64) — все Маки с чёлкой на нём |
| **Тип** | GUI-приложение (`.app`), фоновое (без иконки в доке, `LSUIElement`) |

## 2. Интерфейс (UI)
- **SwiftUI** — весь визуал (пилюля, карточка, чипы, темы, анимации).
- **AppKit** (`NSPanel`, `NSScreen`, `NSEvent`, `Cocoa`) — низкоуровневое окно у чёлки (SwiftUI сам так не умеет). Мост между ними — `NSHostingView`.
- **Combine** (`ObservableObject`, `@Published`) — реактивность: модель поменялась → интерфейс сам перерисовался.

## 3. Сборка и упаковка
- **Swift Package Manager (SPM)** — сборка и зависимости. Это **не** классический Xcode-проект (`.xcodeproj` нет).
- Сборка: `swift build` → бинарь; скрипт `packaging/build_app.sh` заворачивает его в `Miqat.app`.
- Иконка: сгенерирована через `sips` + `iconutil` (`.icns`).

## 4. Сторонние библиотеки (2 штуки)
| Библиотека | Автор / ссылка | Зачем | Лицензия |
|---|---|---|---|
| **Adhan** | `batoulapps/adhan-swift` (1.5.0) | расчёт времён намаза и Киблы на устройстве | MIT |
| **SkyLightWindow** | `Lakr233/SkyLightWindow` (1.0.0) | показ окна на заблокированном экране | открытая (см. репо) |

*(Плюс наш файл `CGSSpace.swift` — под лицензией **MPL-2.0**, взят из Parrot/Boring.Notch.)*

## 5. Системные фреймворки Apple (публичные, встроенные)
| Фреймворк | Зачем |
|---|---|
| **Foundation** | базовые типы, даты, форматтеры, календарь (в т.ч. Хиджра) |
| **SwiftUI** | интерфейс |
| **AppKit / Cocoa** | окно, экран, события мыши |
| **CoreLocation** | геолокация + `CLGeocoder` (координаты → город) |
| **UserNotifications** | локальные напоминания о намазе |
| **Combine** | связка модель ↔ интерфейс |

## 6. Приватные API (недокументированные Apple) — важный момент
Используются **только для размещения окна**, больше нигде:
- **CoreGraphics Services (CGS)** — файл `CGSSpace.swift` → показ виджета на **всех рабочих столах (Spaces)**.
- **SkyLight** — файлы `SkyLightLock.swift` + пакет SkyLightWindow → показ на **локскрине**.

**Что это значит (говори так):** «Приватные API — только для окна, данных не касаются. Из-за них нельзя в Mac App Store, но для **прямой раздачи + нотаризации** это ок (нотаризация — антивирус-скан, а не ревью API). Так же делает Boring.Notch».

## 7. Подпись и распространение
- **Developer ID Application** — сертификат для подписи (аккаунт Apple Developer).
- **Hardened runtime** — включён (требование нотаризации).
- **Нотаризация** — `notarytool` (проверка Apple) + `stapler` (staple ticket).
- Инструменты: `codesign`, `notarytool`, `stapler`, `hdiutil` (`.dmg`).
- Один командой: `bash packaging/release.sh` → нотаризованный `Miqat.dmg`.
- Раздача: **GitHub Releases** (прямая ссылка на `.dmg`). **Не App Store.**

## 8. Архитектура (паттерн)
Слоями (≈ MVVM), «каждый знает только соседа»:
```
PrayerEngine        расчёт (Adhan) — данные
   ↑
ClockModel + NotchState   состояние (Combine/ObservableObject)
   ↑
NotchView (SwiftUI)  интерфейс
   ↕
NotchController + CGSSpace + SkyLight   окно у чёлки (AppKit + приватные API)
```
Файлы: `MiqatApp`, `NotchController`, `NotchView`, `PrayerEngine`, `ClockModel`, `Theme`, `Design`, `LocationProvider`, `NotificationScheduler`, `CGSSpace`, `SkyLightLock`.

## 9. Чего НЕТ (частый вопрос разработчиков)
- **Нет сервера / бэкенда / базы данных** — всё на устройстве.
- **Нет аналитики / трекинга / рекламы.**
- **Нет интернета для работы** — оффлайн (сеть нужна лишь геокодеру для названия города, один раз).

## 10. Инфраструктура / релиз
- Репозиторий: **`github.com/vosayll/miqat`** (публичный).
- Домен: **miqat.space** (регистратор **reg.ru**), хостинг лендинга — **Sweb / SpaceWeb**.
- Дистрибутив: `Miqat.dmg` в GitHub Releases.

## Мини-глоссарий (чтобы понимать, о чём говорят)
- **Swift** — язык Apple. **SwiftUI** — современный способ рисовать интерфейс. **AppKit** — старый низкоуровневый UI для macOS.
- **SPM (Swift Package Manager)** — система сборки и зависимостей.
- **ObservableObject / @Published (Combine)** — «данные меняются → UI сам обновляется».
- **Приватный API** — недокументированная функция Apple (работает, но не для App Store).
- **Developer ID** — сертификат для раздачи вне App Store. **Hardened runtime** — режим безопасности приложения.
- **Нотаризация** — проверка приложения Apple на вредоносность (для «чистой» установки). **Gatekeeper** — защита macOS, которая это проверяет.
- **.dmg** — образ-контейнер для установки Mac-приложения. **GitHub Releases** — раздача файлов через GitHub.
- **Чёлка / safe area / notch** — вырез сверху экрана MacBook, вокруг которого живёт виджет.
