# Техдолг Miqat

Отложенное — вернуться, когда дойдут руки.

## Открыто

- **Отдельное окно настроек.** При запуске приложения показывать **отдельное окно настроек** пилюли (обычное окно, НЕ сама чёлка-панель). В нём: город / метод расчёта / мазхаб, звук азана, показ на локскрине, формат времени (12/24ч) и пр. Сейчас настроек нет — всё зашито в код. Появится — вынести туда переключатели уже сделанных фич.

- **Крайне высокие широты (полярный день/ночь).** За полярным кругом (напр. Мурманск, 69°N) летом солнце не заходит → времена не вычисляются, расписание пустое. Правило «седьмой ночи» спасает Москву/Лондон у солнцестояния, но не полярный день. Решение позже: метод «ближайшего города/широты» (напр. брать времена Мекки или ближайшей нормальной широты) или ручной выбор правила в настройках.

- **Геолокация/геокодер для чужих городов не протестированы вживую.** Проверена только логика координаты→времена (7 городов). Реальный GPS-фикс в другом городе и название города из `CLGeocoder` не гонялись — нельзя физически переместить Мак. Риск низкий (API Apple), но факт.
